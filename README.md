# wildberries-ethical-consumption-analysis

# Цифровой след этичного потребления на Wildberries

Эконометрический анализ влияния эко-маркировки на цену и потребительский рейтинг товаров 
на крупнейшем российском маркетплейсе Wildberries.

## Цель исследования

Количественно оценить, существует ли ценовая премия за экологическую маркировку 
(«эко», «био», «органик») и как она варьируется между товарными категориями 
(продукты питания, косметика, бытовая химия).

## Данные

- **Источник:** каталог Wildberries (веб-скрейпинг топ-1000 органической выдачи)
- **Объём:** 3074 товара после очистки
- **Период:** апрель 2026 г. (кросс-секционные данные)
- **Категории:** food (1440), cosmetics (1008), chemistry (626)
- **Доля эко-товаров:** 5.0% (172 единицы)
